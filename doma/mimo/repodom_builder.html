📖 Jak Repo dům funguje?

🎯 URL otisk — jak sdílíš bez serveru? ▼

URL otisk je tvůj supermoc. Místo toho aby si data uložila na server, jsou v URL:

https://tvuj-projekt.github.io/katalog.html?typ=ptak&habitat=les&barva=modra

Když pošleš tohle URL:

  • Nikdo si data nemusí pamatovat na serveru
  • Přítele vidí přesně to co ty
  • Je to permanentní odkaz bez databáze
  • Offline? Funguje stejně (jen URL bez nových dat)

Jak to udělat v kódu:

// Přečti URL parametry const params = new URLSearchParams(window.location.search); const typ = params.get('typ') || 'vsechny'; const habitat = params.get('habitat') || ''; // Postav URL otisk const url = new URL(window.location); url.searchParams.set('typ', vybranyTyp); url.searchParams.set('habitat', vybranyHabitat); window.history.replaceState(null, '', url); // Pošli lidem tento odkaz — je permanentní a nesdílí data
💾 localStorage — tvůj offline úschovný ▼

localStorage je místo v tvém prohlížeči kde si můžeš uchovávat až 5-10 MB dat.

Uložit data:

const moje_data = { jmeno: 'Jan', poznamka: 'Zajímavý ptáček' }; localStorage.setItem('zaznam_5', JSON.stringify(moje_data));

Přečíst data:

const data = JSON.parse(localStorage.getItem('zaznam_5')); console.log(data.jmeno); // Jan

Smazat data:

localStorage.removeItem('zaznam_5'); localStorage.clear(); // Všechno

⚠️ Důležité: localStorage je jen pro tvůj počítač. Když si stránku otevřeš na jiném počítači, data tam nejsou. Proto musíš uživateli umožnit exportovat data jako JSON/CSV.

📐 Konzistentní cesty — proč je to důležité? ▼

Všechny tvé projekty mají stejnou strukturu. Proč?

  • Přenositelnost: Vezmeš si projekt, dáš na jiný server, funguje
  • Sdílení assetů: Ilustrace jsou v doma/ilustrace/ — všechny projekty je najdou
  • Bez zmatení: Vždy víš kde co je
  • Modulárnost: Můžeš kopírovat nástroje mezi projekty
projekt-1/ ├── doma/ │ ├── ilustrace/ │ ├── nastroje/ │ └── data/ projekt-2/ ├── doma/ │ ├── ilustrace/ │ ├── nastroje/ │ └── data/ Cesty jsou **stejné**, takže když si vezmeš obrázek z projektu-1, stačí zkopírovat URL — projekt-2 jej najde na stejném místě.
🔄 Export a synchronizace ▼

Jak uživatel exportuje svá data bez serveru?

Stažení jako JSON:

function exportJson() { const data = localStorage.getItem('moje_data'); const blob = new Blob([data], { type: 'application/json' }); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'backup_' + new Date().toISOString() + '.json'; a.click(); }

Stažení jako CSV:

function exportCsv() { const data = JSON.parse(localStorage.getItem('moje_data')); const csv = Object.keys(data[0]).join(',') + '\n' + data.map(row => Object.values(row).join(',')).join('\n'); const blob = new Blob([csv], { type: 'text/csv' }); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'export.csv'; a.click(); }

Import zpátky:

document.getElementById('fileInput').addEventListener('change', (e) => { const file = e.target.files[0]; const reader = new FileReader(); reader.onload = (event) => { const data = JSON.parse(event.target.result); localStorage.setItem('moje_data', JSON.stringify(data)); location.reload(); // Znovu načti UI }; reader.readAsText(file); });
